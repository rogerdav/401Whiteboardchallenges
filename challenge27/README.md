#27 Rotate a 2D array 90 Degrees
## Big O(n)

Given a 2-dimensional array of numbers, rotate the array 90 degrees.

[
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9],
]

// rotated 90 degrees
[
  [7, 4, 1],
  [8, 5, 2],
  [9, 6, 3],
]
